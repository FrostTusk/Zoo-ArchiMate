Zoo Architecture
An ArchiMate model for Zoo, a conceptual custom IT service provider. This model is used to: 1) Maintain an architecture of my personal IT initiatives for documentation purposes. 2) Practice ArchiMate modeling to develop and maintain my modeling skills. 3) Educate and spread awareness on the importance of enterprise architecture and how to use the ArchiMate framework. Ocassionaly an element with this diagram might be tagged using "<>", this is to visually convey meta-information. Tag dictionary: {"D"=deprecated, "R"=redacted, "E"=elsewhere}. The Capability Map is a good starting point to follow along and understand the model. *This project is licensed under the MIT license*
